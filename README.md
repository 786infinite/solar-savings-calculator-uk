# Solar Savings Calculator UK

Beginner-friendly Next.js project for a UK solar calculator and affiliate website.

## How to run locally

1. Open this folder in VS Code.
2. Open Terminal > New Terminal.
3. Run:

```bash
npm install
npm run dev
```

4. Open http://localhost:3000

## Main pages included

- Homepage
- Solar Savings Calculator
- Affiliate placeholder redirect pages
- Affiliate Disclosure
- Privacy Policy
- Cookie Policy
- Terms
- Contact
- Calculator Methodology
- 10 SEO starter pages
